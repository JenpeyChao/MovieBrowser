What is this project?

This is a movie browising application inspired by IMDB. The App uses the OMDB API to gather details about movies. The movies are acquired by users typing in specific movie titles and release dates. We save the data retrieved from the API into a local json-server and allow the user to see their search history so they can always view what they have searched before.
